Autodesk Inventor CAD Joint Translation: Export joints defined in CAD assembly using Simscape Multibody Link

Export Autodesk Inventor CAD assembly models containing joint definitions. All joint types are supported: Rigid, Rotational, Slider, Cylindrical, Planar, and Ball. Previously, only assembly constraints were supported. For more information, see CAD Translation and Constraints and Joints.

CAD Export to Simscape Multibody First Generation Software Deprecated

The Simscape™ Multibody™ Link now exports CAD assemblies only in an XML format compatible with Simscape Multibody Second Generation software. All models imported from such XML files are created from second-generation blocks—those accessible by entering the command smlib at the MATLAB® command prompt. The plug-in no longer provides the option to export in an XML file compatible with first-generation blocks—those accessible by entering the command mech_lib at the MATLAB command prompt. You can continue to import older XML files originally meant for translation into first-generation models using the mech_import function.

SimMechanics Link renamed to Simscape Multibody Link

The SimMechanics™ Link CAD plug-in is now named Simscape Multibody Link. The plug-in enables you to export a CAD assembly from a supported CAD application into what is now named Simscape Multibody software.
