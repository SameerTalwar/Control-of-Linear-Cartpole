smlink_unlinksw

Remove Simscape Multibody Link from your Windows registry as a SolidWorks plugin

Syntax

smlink_unlinksw

Description

smlink_unlinksw removes Simscape™ Multibody™ Link from your Windows® registry as a SolidWorks® plugin. The plugin becomes no longer discoverable in your SolidWorks installation. You can no longer export SolidWorks assembly models for import into the Simscape Multibody environment. Use the smlink_linksw command if you decide to once again register the plugin and continue to use its features.
