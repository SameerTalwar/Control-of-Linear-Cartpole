PmitLengthUnit

Enumerated type for specifying length unit in API session

Description

Warning

This content is specific to Simscape™ Multibody™ First Generation software. First-generation features are slated to be deprecated and should be avoided.

PmitLengthUnit is a C language enumerated type.

You can define a variable of this type globally when you start a Simscape Multibody Link API session.

These are the variable's allowed enumerated values.

ValueLength Unit Type
PMIT_LU_UNKNOWN = -1Unknown
PMIT_LU_M = 0Meter
PMIT_LU_CMCentimeter
PMIT_LU_MMMillimeter
PMIT_LU_KMKilometer
PMIT_LU_INInch
PMIT_LU_FTFoot
PMIT_LU_MIMile
PMIT_LU_YDYard
